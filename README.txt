EASILY CREATE A BRIGHT SLIDEOUT MENU

== Description ==
Create the bright slide-out menu with the "Slide Menu" plugin for the Wordpress sites. To open just click on the compact hamburger icon in the page corner. It can contain the large number of navigation elements and options.  

= Demo =
[Demo](https://wow-estore.com/item-preview/slide-menu/)

[youtube https://www.youtube.com/watch?v=Viy3N3w-rFw]

= Main features =
* Create 1 slideout menu
* Left Position of menu
* 2 Positions of menu button
* Add the awesome icons to each element
* Insert the links and other navigation elements


= Slide Menu can be used for: = 
* Site navigation
* Contacts block
* User profile navigation
* Skype menu
* And more...

== Installation ==
* Installation option 1: Find and install this plugin in the `Plugins` -> `Add new` section of your `wp-admin`
* Installation option 2: Download the zip file, then upload the plugin via the wp-admin in the `Plugins` -> `Add new` section. Or unzip the archive and upload the folder to the plugins directory `/wp-content/plugins/` via ftp
* Press `Activate` when you have installed the plugin via dashboard or press `Activate` in the in the `Plugins` list 
* Go to `Slide Menu` section that will appear in your main menu on the left
* Click `Add new` to create your first menu
* Setup your menu
* Click Save
